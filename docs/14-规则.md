# 规则（AGENTS.md）

通过创建 `AGENTS.md` 文件为 OpenCode 设置自定义指令。

## 初始化

```bash
/init
```

扫描项目并生成 `AGENTS.md` 文件。

## 文件位置

### 项目级

项目根目录的 `AGENTS.md` - 仅在该目录工作时生效。

### 全局级

`~/.config/opencode/AGENTS.md` - 应用于所有会话。

### Claude Code 兼容性

- `CLAUDE.md`（项目）
- `~/.claude/CLAUDE.md`（全局）

## 优先级

1. 本地文件（向上遍历）
2. 全局文件 `~/.config/opencode/AGENTS.md`
3. Claude Code 文件 `~/.claude/CLAUDE.md`

## 示例

```markdown
# SST v3 Monorepo Project

## Project Structure
- `packages/` - 工作区包
- `infra/` - 基础设施定义
- `sst.config.ts` - SST 配置

## Code Standards
- 使用 TypeScript 严格模式
- 共享代码放在 `packages/core/`
```

## 自定义指令文件

```json
{
  "instructions": [
    "CONTRIBUTING.md",
    "docs/guidelines.md",
    ".cursor/rules/*.md"
  ]
}
```

## 文件引用

```markdown
# AGENTS.md

## Guidelines
For React patterns: @docs/react-patterns.md
```

OpenCode 支持在 AGENTS.md 中使用 `@` 引用文件。
