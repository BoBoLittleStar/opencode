# 主题

选择内置主题或创建自定义主题。

## 内置主题

| 主题 | 说明 |
|------|------|
| `opencode` | 默认主题 |
| `system` | 适配终端配色 |
| `tokyonight` | Tokyo Night |
| `everforest` | Everforest |
| `ayu` | Ayu 暗色 |
| `catppuccin` | Catppuccin |
| `catppuccin-macchiato` | Catppuccin Macchiato |
| `gruvbox` | Gruvbox |
| `kanagawa` | Kanagawa |
| `nord` | Nord |
| `matrix` | 黑客风格 |
| `one-dark` | One Dark |

## 使用主题

```bash
/themes
```

或在 `tui.json` 中指定：

```json
{
  "theme": "tokyonight"
}
```

## 自定义主题

主题位置（优先级从低到高）：

1. 内置主题
2. `~/.config/opencode/themes/*.json`
3. `<project>/.opencode/themes/*.json`
4. `./.opencode/themes/*.json`

### 创建主题

`~/.config/opencode/themes/my-theme.json`：

```json
{
  "$schema": "https://opencode.ai/theme.json",
  "defs": {
    "nord8": "#88C0D0"
  },
  "theme": {
    "primary": {
      "dark": "nord8"
    },
    "text": {
      "dark": "#D8DEE9"
    },
    "background": {
      "dark": "#2E3440"
    }
  }
}
```

## 颜色定义

| 属性 | 说明 |
|------|------|
| `primary` | 主色 |
| `secondary` | 次要色 |
| `accent` | 强调色 |
| `error` | 错误色 |
| `warning` | 警告色 |
| `success` | 成功色 |
| `info` | 信息色 |
| `text` | 文本色 |
| `textMuted` | 次要文本色 |
| `background` | 背景色 |
| `backgroundPanel` | 面板背景 |
| `diffAdded` | 差异添加 |
| `diffRemoved` | 差异删除 |

## 颜色格式

```json
{
  "primary": "#88C0D0",
  "primary": 8,  // ANSI 颜色
  "primary": "accent",  // 引用
  "primary": {
    "dark": "#88C0D0",
    "light": "#5E81AC"
  }
}
```

## 终端默认值

使用 `"none"` 使用终端默认颜色：

```json
{
  "text": "none",
  "background": "none"
}
```

## 系统主题

`system` 主题自动适配终端配色：

- 根据终端背景颜色生成灰度色阶
- 使用 ANSI 颜色
- 保留终端原生外观
