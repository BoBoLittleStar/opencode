# 模型配置

OpenCode 使用 [AI SDK](https://ai-sdk.dev/) 和 [Models.dev](https://models.dev) 支持 **75+ LLM 提供商**，并支持运行本地模型。

## 选择模型

配置好提供商后，使用 `/models` 命令选择模型：

```bash
/models
```

## 推荐模型

市面上有非常多的模型，但真正擅长代码生成和工具调用的模型只有少数几个。以下是推荐列表：

| 模型 | 提供商 | 特点 |
|------|--------|------|
| GPT 5.2 | OpenAI | 最新 GPT 模型 |
| GPT 5.1 Codex | OpenAI | 专为编程优化 |
| Claude Opus 4.5 | Anthropic | 最强性能 |
| Claude Sonnet 4.5 | Anthropic | 平衡性能与成本 |
| MiniMax M2.1 | MiniMax | 高性价比 |
| Gemini 3 Pro | Google | 强大的推理能力 |

## 设置默认模型

```json
{
  "model": "lmstudio/google/gemma-3n-e4b"
}
```

模型 ID 格式为 `provider_id/model_id`，例如：
- `openai/gpt-5.1-codex`
- `anthropic/claude-sonnet-4-5`
- `opencode/gpt-5.3-codex`（使用 Zen 时）

## 配置模型选项

```jsonc
{
  "provider": {
    "openai": {
      "models": {
        "gpt-5": {
          "options": {
            "reasoningEffort": "high",
            "textVerbosity": "low",
            "reasoningSummary": "auto",
            "include": ["reasoning.encrypted_content"]
          }
        }
      }
    },
    "anthropic": {
      "models": {
        "claude-sonnet-4-5-20250929": {
          "options": {
            "thinking": {
              "type": "enabled",
              "budgetTokens": 16000
            }
          }
        }
      }
    }
  }
}
```

## 模型变体

### 内置变体

**Anthropic**：
- `high` - 高思考预算（默认）
- `max` - 最大思考预算

**OpenAI**：
- `none` - 无推理
- `minimal` - 极少推理
- `low` - 低推理
- `medium` - 中等推理
- `high` - 高推理
- `xhigh` - 超高推理

**Google**：
- `low` - 较低推理
- `high` - 较高推理

### 自定义变体

```jsonc
{
  "provider": {
    "openai": {
      "models": {
        "gpt-5": {
          "variants": {
            "thinking": {
              "reasoningEffort": "high",
              "textVerbosity": "low"
            },
            "fast": {
              "disabled": true
            }
          }
        }
      }
    }
  }
}
```

### 切换变体

使用快捷键 `variant_cycle` 循环切换变体。

## 加载模型优先级

1. `--model` / `-m` 命令行标志
2. 配置文件中的 `model` 字段
3. 上次使用的模型
4. 按内部优先级排列的第一个可用模型

## 小模型配置

用于标题生成等轻量级任务：

```json
{
  "small_model": "anthropic/claude-haiku-4-5"
}
```

## 模型限制

```json
{
  "models": {
    "my-model": {
      "limit": {
        "context": 128000,
        "output": 65536
      }
    }
  }
}
```

## 环境变量

| 变量 | 说明 |
|------|------|
| `OPENCODE_ENABLE_EXPERIMENTAL_MODELS` | 启用实验性模型 |
| `OPENCODE_MODELS_URL` | 自定义模型配置获取 URL |
