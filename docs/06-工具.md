# 工具

工具允许 LLM 在代码库中执行操作。OpenCode 自带内置工具，也可通过自定义工具和 MCP 服务器扩展。

## 权限控制

```json
{
  "permission": {
    "edit": "allow",
    "bash": "ask"
  }
}
```

权限值：`"allow"` | `"ask"` | `"deny"`

详见 [17-权限.md](./17-权限.md)

## 内置工具

### bash

执行 shell 命令。

```json
{
  "permission": {
    "bash": "allow"
  }
}
```

### edit

通过精确字符串替换修改文件。

```json
{
  "permission": {
    "edit": "allow"
  }
}
```

### write

创建新文件或覆盖现有文件。由 `edit` 权限控制。

### read

读取文件内容，支持行范围。

```json
{
  "permission": {
    "read": "allow"
  }
}
```

### grep

使用正则表达式搜索文件内容。

```json
{
  "permission": {
    "grep": "allow"
  }
}
```

### glob

通过模式匹配查找文件。

```json
{
  "permission": {
    "glob": "allow"
  }
}
```

### list

列出目录内容。

```json
{
  "permission": {
    "list": "allow"
  }
}
```

### lsp（实验性）

与 LSP 服务器交互。需要 `OPENCODE_EXPERIMENTAL_LSP_TOOL=true`。

```json
{
  "permission": {
    "lsp": "allow"
  }
}
```

支持操作：`goToDefinition`、`findReferences`、`hover`、`documentSymbol`、`workspaceSymbol` 等。

### patch

对文件应用补丁。由 `edit` 权限控制。

### skill

加载技能文件并返回内容。

```json
{
  "permission": {
    "skill": "allow"
  }
}
```

### todowrite

管理待办事项列表。

```json
{
  "permission": {
    "todowrite": "allow"
  }
}
```

### webfetch

获取网页内容。

```json
{
  "permission": {
    "webfetch": "allow"
  }
}
```

### websearch

在网络上搜索信息。需要 `OPENCODE_ENABLE_EXA=true`。

```json
{
  "permission": {
    "websearch": "allow"
  }
}
```

### question

在执行过程中向用户提问。

```json
{
  "permission": {
    "question": "allow"
  }
}
```

## 忽略模式

`grep`、`glob`、`list` 底层使用 ripgrep，默认遵循 `.gitignore` 中的模式。

创建 `.ignore` 文件可包含通常被忽略的路径：

```gitignore
!node_modules/
!dist/
!build/
```

## 工具命名空间

MCP 服务器的工具以服务器名为前缀，如 `mcp_github_search_repos`。

使用 glob 模式控制多个工具：

```json
{
  "permission": {
    "mymcp_*": "ask"
  }
}
```
