# 安装 OpenCode

## Linux/macOS 安装

### 安装脚本（推荐）

```bash
curl -fsSL https://opencode.ai/install | bash
```

### Node.js 包管理器

```bash
# npm
npm install -g opencode-ai

# bun
bun install -g opencode-ai

# pnpm
pnpm install -g opencode-ai

# yarn
yarn global add opencode-ai
```

### Homebrew（macOS/Linux）

```bash
brew install anomalyco/tap/opencode
```

> 推荐使用 OpenCode tap 以获取最新版本。

### Arch Linux

```bash
# 稳定版
sudo pacman -S opencode

# AUR 最新版
paru -S opencode-bin
```

## Windows 安装

### 方式一：WSL（推荐）

为获得最佳体验，推荐使用 Windows Subsystem for Linux：

```bash
# 安装 WSL
wsl --install

# 在 WSL 中安装 OpenCode
curl -fsSL https://opencode.ai/install | bash
```

详见 [29-Windows(WSL).md](./29-Windows(WSL).md)

### 方式二：Chocolatey

```bash
choco install opencode
```

### 方式三：Scoop

```bash
scoop install opencode
```

### 方式四：NPM

```bash
npm install -g opencode-ai
```

### 方式五：Mise

```bash
mise use -g github:anomalyco/opencode
```

### 方式六：Docker

```bash
docker run -it --rm ghcr.io/anomalyco/opencode
```

### 方式七：直接下载

从 [Releases](https://github.com/anomalyco/opencode/releases) 页面下载二进制文件。

## 验证安装

安装完成后，验证安装：

```bash
opencode --version
```

## 卸载

```bash
opencode uninstall
```

### 卸载选项

| 选项 | 说明 |
|------|------|
| `--keep-config` | 保留配置文件 |
| `--keep-data` | 保留会话数据和快照 |
| `--dry-run` | 显示将被删除的内容 |
| `--force` | 跳过确认提示 |

## 升级

```bash
# 升级到最新版本
opencode upgrade

# 升级到指定版本
opencode upgrade v0.1.48

# 指定安装方式
opencode upgrade --method npm
```

## 终端配置

OpenCode 对终端有特定要求以确保最佳体验：

### 真彩色支持

主题需要真彩色（24 位色）支持。检查终端支持：

```bash
echo $COLORTERM
# 应输出: truecolor 或 24bit
```

### Linux 剪贴板工具

Linux 用户需要安装剪贴板工具：

```bash
# X11
apt install -y xclip
# 或
apt install -y xsel

# Wayland
apt install -y wl-clipboard

# 无头环境
apt install -y xvfb
Xvfb :99 -screen 0 1024x768x24 > /dev/null 2>&1 &
export DISPLAY=:99.0
```
