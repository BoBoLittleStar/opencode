# 自定义工具

创建 LLM 可调用的自定义工具。

## 文件位置

- 项目级：`.opencode/tools/`
- 全局：`~/.config/opencode/tools/`

## 基本结构

```typescript
import { tool } from "@opencode-ai/plugin"

export default tool({
  description: "Query the project database",
  args: {
    query: tool.schema.string().describe("SQL query to execute"),
  },
  async execute(args) {
    return `Executed query: ${args.query}`
  },
})
```

文件名即为工具名称，上例创建了 `database` 工具。

## 参数定义

使用 `tool.schema`（Zod）：

```typescript
args: {
  query: tool.schema.string().describe("SQL query"),
  limit: tool.schema.number().optional(),
}
```

或直接导入 Zod：

```typescript
import { z } from "zod"

export default {
  description: "Tool description",
  args: {
    param: z.string().describe("Parameter description"),
  },
  async execute(args, context) {
    return "result"
  },
}
```

## 上下文

工具接收会话上下文：

```typescript
async execute(args, context) {
  const { agent, sessionID, messageID, directory, worktree } = context
  return `Agent: ${agent}, Session: ${sessionID}`
}
```

## 多工具导出

单文件可导出多个工具：

```typescript
export const add = tool({
  description: "Add two numbers",
  args: {
    a: tool.schema.number(),
    b: tool.schema.number(),
  },
  async execute(args) {
    return args.a + args.b
  },
})

export const multiply = tool({
  description: "Multiply two numbers",
  args: {
    a: tool.schema.number(),
    b: tool.schema.number(),
  },
  async execute(args) {
    return args.a * args.b
  },
})
```

创建工具：`math_add` 和 `math_multiply`。

## 替换内置工具

文件名与内置工具同名会替换内置工具，如 `.opencode/tools/bash.ts` 替换内置 bash 工具。

## 使用其他语言

工具定义用 TypeScript/JavaScript，但可调用任何语言的脚本：

### Python 示例

创建 Python 脚本：

`.opencode/tools/add.py`：
```python
import sys
a = int(sys.argv[1])
b = int(sys.argv[2])
print(a + b)
```

创建工具定义：

```typescript
import { tool } from "@opencode-ai/plugin"
import path from "path"

export default tool({
  description: "Add two numbers using Python",
  args: {
    a: tool.schema.number(),
    b: tool.schema.number(),
  },
  async execute(args, context) {
    const script = path.join(context.worktree, ".opencode/tools/add.py")
    const result = await Bun.$`python3 ${script} ${args.a} ${args.b}`.text()
    return result.trim()
  },
})
```

## 依赖项

在配置目录创建 `package.json`：

`.opencode/package.json`：
```json
{
  "dependencies": {
    "shescape": "^2.1.0"
  }
}
```

OpenCode 会自动运行 `bun install`。

然后在插件中导入：

```typescript
import { escape } from "shescape"
```
