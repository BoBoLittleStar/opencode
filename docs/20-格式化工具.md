# 格式化工具

OpenCode 在文件写入或编辑后自动格式化代码。

## 内置格式化工具

| 格式化工具 | 扩展名 | 要求 |
|------------|--------|------|
| air | .R | `air` 命令 |
| biome | .js, .jsx, .ts, .tsx, .html, .css... | `biome.json` 配置 |
| cargofmt | .rs | `cargo fmt` |
| clang-format | .c, .cpp, .h, .hpp... | `.clang-format` 配置 |
| cljfmt | .clj, .cljs, .cljc, .edn | `cljfmt` 命令 |
| dart | .dart | `dart` 命令 |
| dfmt | .d | `dfmt` 命令 |
| gofmt | .go | `gofmt` 命令 |
| ktlint | .kt, .kts | `ktlint` 命令 |
| mix | .ex, .exs, .eex... | `mix` 命令 |
| nixfmt | .nix | `nixfmt` 命令 |
| ocamlformat | .ml, .mli | `ocamlformat` 命令 |
| ormolu | .hs | `ormolu` 命令 |
| pint | .php | `laravel/pint` |
| prettier | .js, .jsx, .ts, .tsx... | `prettier` 依赖 |
| rubocop | .rb, .rake... | `rubocop` 命令 |
| ruff | .py, .pyi | `ruff` 命令 |
| rustfmt | .rs | `rustfmt` 命令 |
| shfmt | .sh, .bash | `shfmt` 命令 |
| terraform | .tf, .tfvars | `terraform` 命令 |

## 工作原理

1. 检查文件扩展名
2. 运行相应的格式化命令
3. 自动应用更改

## 配置

```json
{
  "formatter": {
    "prettier": {
      "disabled": true
    }
  }
}
```

## 禁用所有格式化工具

```json
{
  "formatter": false
}
```

## 自定义格式化工具

```json
{
  "formatter": {
    "custom-prettier": {
      "command": ["npx", "prettier", "--write", "$FILE"],
      "environment": {
        "NODE_ENV": "development"
      },
      "extensions": [".js", ".ts", ".jsx", ".tsx"]
    }
  }
}
```

`$FILE` 占位符会被替换为待格式化的文件路径。

## 格式化工具属性

| 属性 | 类型 | 说明 |
|------|------|------|
| `disabled` | boolean | 禁用 |
| `command` | string[] | 格式化命令 |
| `environment` | object | 环境变量 |
| `extensions` | string[] | 文件扩展名 |
