# 快捷键

自定义 OpenCode 快捷键。

## 配置文件

`tui.json`：

```json
{
  "$schema": "https://opencode.ai/tui.json",
  "keybinds": {
    "leader": "ctrl+x",
    "session_new": "<leader>n",
    "session_list": "<leader>l"
  }
}
```

## 前导键

默认 `ctrl+x` 是前导键，使用前需先按前导键再按快捷键。

如 `ctrl+x n` 表示：先按 `ctrl+x`，再按 `n`。

## 禁用快捷键

```json
{
  "keybinds": {
    "session_compact": "none"
  }
}
```

## 常用快捷键

| 快捷键 | 说明 |
|--------|------|
| `Tab` | 切换代理 |
| `ctrl+x n` | 新建会话 |
| `ctrl+x l` | 会话列表 |
| `ctrl+x m` | 模型列表 |
| `ctrl+x t` | 主题列表 |
| `ctrl+x c` | 压缩会话 |
| `ctrl+x s` | 分享会话 |
| `ctrl+x h` | 帮助 |
| `ctrl+x q` | 退出 |
| `ctrl+x u` | 撤销 |
| `ctrl+x r` | 重做 |
| `ctrl+x d` | 工具详情 |
| `ctrl+x e` | 打开编辑器 |
| `ctrl+x i` | 初始化 |
| `ctrl+x x` | 导出会话 |

## 滚动快捷键

| 快捷键 | 说明 |
|--------|------|
| `pageup` / `pagedown` | 翻页 |
| `ctrl+alt+y` / `ctrl+alt+e` | 单行滚动 |
| `ctrl+alt+u` / `ctrl+alt+d` | 半页滚动 |
| `home` / `end` | 首/末页 |

## 输入快捷键

| 快捷键 | 说明 |
|--------|------|
| `return` | 提交 |
| `shift+return` | 换行 |
| `ctrl+a` / `ctrl+e` | 行首/行尾 |
| `alt+b` / `alt+f` | 词前进/退 |
| `ctrl+k` | 删除到行尾 |
| `ctrl+u` | 删除到行首 |
| `ctrl+w` | 删除前词 |

## Windows Terminal Shift+Enter

在 `settings.json` 添加：

```json
"actions": [
  {
    "command": {
      "action": "sendInput",
      "input": "\u001b[13;2u"
    },
    "id": "User.sendInput.ShiftEnterCustom"
  }
]
```
