# OpenCode Go

低成本的开源编程模型订阅服务。

## 概述

**首月 $5**，之后 **每月 $10**，访问热门开源编程模型。

> Go 完全可选，与其他提供商使用方式相同。

## 支持的模型

- **GLM-5**
- **Kimi K2.5**
- **MiniMax M2.7**
- **MiniMax M2.5**

## 使用限制

| 限制 | 额度 |
|------|------|
| 5 小时限制 | $12 使用额度 |
| 每周限制 | $30 使用额度 |
| 每月限制 | $60 使用额度 |

### 预估请求数

| 模型 | 每 5 小时 | 每周 | 每月 |
|------|-----------|------|------|
| GLM-5 | 1,150 | 2,880 | 5,750 |
| Kimi K2.5 | 1,850 | 4,630 | 9,250 |
| MiniMax M2.5 | 20,000 | 50,000 | 100,000 |

## API 端点

```bash
# 端点
https://opencode.ai/zen/go/v1/chat/completions
https://opencode.ai/zen/go/v1/messages

# 模型 ID 格式
opencode-go/<model-id>
```

## 超出限制

Zen 余额有积分时，可启用 "Use balance" 选项。

## 隐私

- 模型托管在 US、EU、Singapore
- 提供商遵循零保留政策
- 数据不用于训练

## 目标

1. 低成本订阅，无门槛使用 AI 编程
2. 可靠访问最佳开源模型
3. 精选适合编程 Agent 的模型
4. 无锁定，可与其他提供商一起使用
