# 分享功能

OpenCode 允许创建指向对话的公开链接，方便与团队协作或向他人求助。

## 分享模式

| 模式 | 说明 |
|------|------|
| `manual`（默认） | 使用 `/share` 手动分享 |
| `auto` | 新会话自动分享 |
| `disabled` | 完全禁用分享 |

### 手动分享（默认）

```bash
/share
```

生成唯一 URL 并复制到剪贴板。

### 自动分享

```json
{
  "share": "auto"
}
```

### 禁用分享

```json
{
  "share": "disabled"
}
```

## 取消分享

```bash
/unshare
```

移除分享链接并删除相关数据。

## 隐私注意

- 共享对话对任何拥有链接的人公开
- 对话默认不分享
- 避免分享敏感信息
- 协作完成后请取消分享

## 企业版

企业部署可以：
- 完全禁用分享功能
- 限制为仅 SSO 用户可用
- 自托管分享服务
