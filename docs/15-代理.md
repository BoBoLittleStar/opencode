# 代理

代理是专门的 AI 助手，可以针对特定任务和工作流程进行配置。

## 代理类型

### 主代理

直接交互的主要助手，使用 `Tab` 键或 `switch_agent` 快捷键切换。

- **Build**（默认）- 启用了所有工具
- **Plan** - 规划和分析，无修改权限

### 子代理

主代理可调用的专业助手，通过 `@` 提及或自动调用：

- **General** - 通用研究和多步骤任务
- **Explore** - 快速只读代码探索

## 内置代理

| 代理 | 模式 | 说明 |
|------|------|------|
| Build | primary | 默认开发代理，所有工具 |
| Plan | primary | 规划模式，文件/bash 需审批 |
| General | subagent | 通用研究代理 |
| Explore | subagent | 只读探索代理 |
| Compaction | primary | 系统代理，会话压缩 |
| Title | primary | 系统代理，生成标题 |
| Summary | primary | 系统代理，创建摘要 |

## 配置代理

### JSON 格式

```json
{
  "agent": {
    "code-reviewer": {
      "description": "Reviews code for best practices",
      "mode": "subagent",
      "model": "anthropic/claude-sonnet-4-20250514",
      "prompt": "You are a code reviewer...",
      "tools": {
        "write": false,
        "edit": false
      }
    }
  }
}
```

### Markdown 格式

`.opencode/agents/review.md` 或 `~/.config/opencode/agents/review.md`：

```markdown
---
description: Reviews code for quality
mode: subagent
tools:
  write: false
  edit: false
---

You are in code review mode...
```

## 配置选项

### 描述（必需）

```json
{
  "description": "Reviews code for best practices"
}
```

### 温度

```json
{
  "temperature": 0.1
}
```

- 0.0-0.2：集中确定性
- 0.3-0.5：平衡
- 0.6-1.0：创造多样

### 最大步数

```json
{
  "steps": 5
}
```

### 禁用

```json
{
  "disable": true
}
```

### 模式

```json
{
  "mode": "subagent"  // primary | subagent | all
}
```

### 隐藏

隐藏子代理不显示在 `@` 补全中：

```json
{
  "hidden": true
}
```

### 颜色

```json
{
  "color": "#FF5733"
}
```

或使用主题颜色：`primary`、`secondary`、`accent`、`success`、`warning`、`error`、`info`

### Top P

```json
{
  "top_p": 0.9
}
```

## 代理权限

```json
{
  "agent": {
    "build": {
      "permission": {
        "bash": {
          "git *": "ask",
          "git push *": "deny"
        }
      }
    }
  }
}
```

详见 [17-权限.md](./17-权限.md)

## 创建代理

```bash
opencode agent create
```

## 使用场景

- **Build**：完整开发
- **Plan**：分析规划
- **Review**：只读审查
- **Debug**：问题排查
- **Docs**：文档编写
